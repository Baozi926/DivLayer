# DivLayer
[DEMO](http://47.91.224.241:8080/DivLayer/)

## 中文



基于arcgis JavaScript api 4 实现的 div图层，实现可跟随地图移动的div，

因为有了可以随地图移动的div，当我们把echart放在div里，就可以实现一个ehart图层。
